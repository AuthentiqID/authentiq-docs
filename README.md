# Authentiq Documentation

## Developer documentation

Run a local docs server with:

    mkdocs serve

Build the documentation with:

    mkdocs build

Deploy docs to [GitHub pages](http://authentiqid.github.io/authentiq-docs/):

    mkdocs gh-deploy

## API reference documentation
